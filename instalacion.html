Guía de Instalación

Ten Open AURA corriendo en tu servidor en menos de 5 minutos.

🐳

Docker

Recomendado. Un solo comando.

 🐍

Manual

Python + Flutter. Control total sobre el entorno.

01 Opción 1: Docker

Método recomendado para la mayoría de usuarios.

Prerrequisitos

✓

Docker

v20.10 o superior

✓

Docker Compose

v2.0 o superior

Paso 1 Clonar el repositorio

Terminal 
git clone https://github.com/EParache/OpenAura.git
cd OpenAura

Paso 2 Configurar directorio de fotos

Terminal 
# Crea el archivo .env con la ruta a tus fotos
echo "PHOTOS_DIR=/ruta/a/tus/fotos" > .env

Reemplaza /ruta/a/tus/fotos con la carpeta donde guardas tus imágenes y videos.

Paso 3 Iniciar los contenedores

Terminal 
docker compose up -d

Paso 4 Abrir la aplicación

🌐 http://localhost:8000
← Abrí esta URL en tu navegador

02 Opción 2: Manual

Para usuarios que prefieren control total sobre el entorno.

Prerrequisitos

✓

Python

3.10 o superior

✓

Flutter SDK

v3.29 o superior

✓

Git

Cualquier versión

Backend (FastAPI)

1 Clonar y preparar entorno virtual

Terminal 
git clone https://github.com/EParache/OpenAura.git
cd OpenAura/backend

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

2 Iniciar servidor

Terminal 
uvicorn main:app --host 0.0.0.0 --port 8000

Frontend (Flutter)

1 Compilar para web

Terminal 
cd ../frontend
flutter pub get
flutter build web --release

El backend sirve automáticamente los archivos compilados del frontend web.

2 Compilar para escritorio Linux (opcional)

Terminal 
flutter build linux --release
./build/linux/x64/release/bundle/aura_frontend
